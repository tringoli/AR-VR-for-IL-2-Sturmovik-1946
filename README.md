# Welcome!
This guide, created as a repository, is intended to collect all known methods for implementing augmented and virtual reality for the IL-2 Sturmovik: 1946 game. There are many ways to implement AR/VR on Windows, Linux, *and macOS (coming soon™)*.

## What's the problem?
There's practically no information about AR/VR for IL-2. Of course, you can try to figure it out with general software guides, but I wanted to summarize it all for a wide range of IL-2 players.

## Future plans for this guide?

Someone could try creating a special modification for IL-2 to implement native 3D VR rendering. It might be worth considering adapting the game to newer versions of DirectX, which should support VR, etc. However, ["rudiments of VR mod"](https://www.sas1946.com/main/index.php?topic=63444.0) were created in 2020.

## I need your help!
The author can't test every option described here, so you can add important details to the chapters. Feel free to create commits!
