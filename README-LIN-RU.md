# AR/VR для IL-2 Sturmovik: 1946 на Linux
> **:warning: Этот раздел практически не тестировался автором и является скорее теорией использования. Вы можете помочь, отправив отчеты**

Из-за децентрализованности дистрибутивов Linux практически невозможно гарантировать полную совместимость с VR-устройствами. В этом плане Windows — наиболее практичный вариант. С другой стороны, AR не вызывает особых проблем для настройки.

## Прежде чем начать
Реализация AR/VR на Linux не вызовет проблем с вашими важными файлами, но перед следующими шагами не стесняйтесь сделать резервную копию каталога IL-2 Sturmovik.

AR/VR должна работать с модами благодаря различным техническим решениям.

## Адаптация AR для IL-2
Наиболее подходящий путь для тех, кто просто хочет попробовать. Здесь мы представим две возможности: абсолютно бесплатную и с небольшим бюджетом.

### Бюджетная AR

В магазинах/маркетплейсах вашей страны представлено множество AR-устройств. **Будьте аккуратны:** могут возникнуть проблемы с адаптацией IL-2 из-за отсутствия программного обеспечения и/или совместимости для устройства. Список рекомендуемых устройств для покупки не предоставляется, выбор за вами.

#### Что нам нужно?

1. OpenTrack ([GitHub](https://github.com/opentrack/opentrack)) или специализированное программное обеспечение для AR-отслеживания. *Предпочтительнее первый вариант для большей гибкости*

#### Настройка
1. Подготовьте ваше AR-устройство (как описано в инструкции продукта)
2. Откройте OpenTrack. Выберите…
3. Запустите IL-2. В случае успеха внутриигровая камера будет следовать за вашей головой
4. Наслаждайтесь!

### Бесплатная AR
Самый доступный вариант на Linux. Она основана на отслеживании головы с использованием ИИ, встроенного в OpenTrack. Вы можете использовать свой смартфон *или веб-камеру*.

#### Что нам нужно?

1. OpenTrack ([GitHub](https://github.com/opentrack/opentrack))
2. Любое приложение для превращения вашего смартфона в камеру ПК. *Для начала можете попробовать Camo Studio*
3. Для низкой задержки рекомендуется использовать USB-кабель или стабильный Wi-Fi 5G

#### Настройка
1. Подготовьте свой смартфон. Его основная/выбранная камера должна правильно записывать вашу голову с учетом освещения, чтобы избежать неточностей отслеживания
2. Откройте программное обеспечение для работы с камерой как на ПК, так и на смартфоне
3. Откройте opentrack. Выберите…
4. Запустите IL-2. В случае успеха внутриигровая камера будет следовать за вашей головой
5. Наслаждайтесь!

## VR-адаптация для IL-2
Наилучший возможный опыт, но похоже, что **пока нет аналога VorpX, из-за чего полноценная виртуальная реальность невозможна.** 

Возможен костыльный вариант с привязкой окна к взгляду игрока. В теории, через связку SteamVR и OpenTrack (для перевода VR в AR таким образом. У OpenTrack в настройках ввода есть SteamVR). 

В любом случае, для плоского полноэкранного режима существует развилка на ALVR и WiVRn.

### Протестированные VR-гарнитуры
* Meta Quest, Valve Index, другие основные VR-гарнитуры — должны работать
* Pico 4 — работает только ALVR, WiVRn не выводит изображение (зато есть звук)
* Другие менее популярные VR-гарнитуры — возможно

### ALVR
Самый стабильный способ для большинства VR-гарнитур. Он должен работать корректно, если ваша VR-гарнитура не имеет собственного программного обеспечения для преобразования.

#### Что нам нужно?

1. [SteamVR](https://store.steampowered.com/app/250820/SteamVR/)
2. ALVR ([GitHub](https://github.com/alvr-org/ALVR))

#### Настройка
1. Откройте ALVR (как на ПК, так и на VR-гарнитуре) и пройдите настройку
2. Откройте SteamVR
3. Запустите IL-2. В случае успеха вы получите изображение

4. Наслаждайтесь!

### WiVRn

Этот путь предназначен для повышения производительности за счет прямой трансляции IL-2 на VR-гарнитуру.

#### Что нам нужно?

1. WiVRn ([GitHub](https://github.com/WiVRn/WiVRn))

#### Настройка
1. Откройте WiVRn (как на ПК, так и на VR-гарнитуре) и пройдите настройку
2. Запустите Il-2. В случае успеха вы получите изображение

3. Наслаждайтесь!
