# AR/VR для IL-2 Sturmovik: 1946 на Windows
Windows 11 была первой протестированной ОС для эксперимента. Это наиболее предпочтительный путь для AR/VR из-за обширного списка программного обеспечения и т. д.

Она также должна хорошо работать на Windows 10. Windows 8 и более старые версии могут быть слишком устаревшими.

## Прежде чем начать
Реализация AR/VR на Windows не вызовет проблем с вашими важными файлами, но перед следующими шагами не стесняйтесь сделать резервную копию каталога IL-2 Sturmovik.

AR/VR должна работать с модами благодаря разнице в технических решениях.

## Адаптация AR для IL-2
Наиболее подходящий путь для тех, кто просто хочет попробовать. Здесь мы представим две возможности: абсолютно бесплатную и с небольшим бюджетом.

### Бюджетная AR
*Эта глава не тестировалась автором. Вы можете помочь, отправив отчеты.*

В магазинах/маркетплейсах вашей страны представлено множество AR-устройств. У вас не должно возникнуть проблем с адаптацией IL-2 ни к одному из них. Список рекомендуемых устройств для покупки не предоставляется, выбор за вами.

#### Что нам нужно?
1. OpenTrack ([GitHub](https://github.com/opentrack/opentrack)) или специализированное программное обеспечение для AR-отслеживания. *Предпочтительнее первый вариант для большей гибкости*

#### Настройка
1. Подготовьте ваше AR-устройство (как описано в инструкции продукта)
2. Откройте OpenTrack. Выберите удобный источник данных (обычно работает PointTracker)
3. Запустите IL-2. В случае успеха внутриигровая камера будет следовать за вашей головой

4. Наслаждайтесь!

### Бесплатная AR
Она основана на отслеживании головы с использованием ИИ, встроенного в OpenTrack. Вы можете использовать свой смартфон *или веб-камеру*.

#### Что нам нужно?
1. OpenTrack ([GitHub](https://github.com/opentrack/opentrack))
2. Любое приложение для превращения вашего смартфона в камеру ПК. *Для начала можете попробовать Camo Studio*
3. Для низкой задержки рекомендуется использовать USB-кабель или стабильный Wi-Fi 5G

#### Настройка
1. Подготовьте свой смартфон. Его основная/выбранная камера должна правильно записывать вашу голову с учетом освещения, чтобы избежать неточностей отслеживания
2. Откройте программное обеспечение для работы с камерой как на ПК, так и на смартфоне
3. Откройте OpenTrack. Выберите в качестве выходного интерфейса NeuralNet. Зайдите в настройки выхода, выберите камеру и откалибруйтесь
4. Запустите IL-2. В случае успеха внутриигровая камера будет следовать за вашей головой

5. Наслаждайтесь!

## VR-адаптация для IL-2
Наилучший возможный опыт, но требует финансовых вложений в специальное программное обеспечение. Здесь мы будем использовать VorpX для преобразования 2D в 3D VR. Мы опишем пути SteamVR и не-SteamVR.

*Вы можете сделать коммит, если знаете бесплатные варианты. Будет оценено по достоинству*

### Протестированные VR-гарнитуры
* Meta Quest, Valve Index, другие основные VR-гарнитуры — должны работать
* Pico 4 — ✓
* Другие менее популярные VR-гарнитуры — *возможно, отправьте отчеты, чтобы расширить этот список*

### SteamVR
Самый стабильный способ для большинства VR-гарнитур. Он должен работать корректно, если ваша VR-гарнитура не имеет собственного программного обеспечения для преобразования.

#### Что нам нужно?
1. [SteamVR](https://store.steampowered.com/app/250820/SteamVR/)
2. Программа для преобразования вывода с ПК на VR-монитор. *Это может быть любая известная вам программа, например, Virtual Desktop*
3. VorpX. *Вы можете скачать для пробы версию с [4PDA](https://4pda.to/stat/go?u=https%3A%2F%2Fdisk.yandex.ru%2Fd%2F6iYhJOBe3ZtYAg&e=110018648&f=https%3A%2F%2F4pda.to%2Fforum%2Findex.php%3Fshowtopic%3D1034301%26st%3D5260%23entry118062338)*

Дополнительно: профиль IL-2, созданный сообществом на [форуме IL-2](https://forum.il2sturmovik.com/topic/69192-for-rift-s-il2_1946_bat-v385_vorpx_opentrack_logitech-x52_settings_v01/)

#### Настройка
1. Откройте VorpX. Вам нужно выбрать страницу конфигурации ("vorpX Config"). В разделе "General" выберите "SteamVR (Index, Vive, Pimax)"
2. В разделе "Cloud/Local Profiles" скачайте онлайн или установите созданный сообществом профиль для IL-2
3. Откройте программу для ретрансляции (обычно как на ПК, так и на VR-гарнитуре)
4. Откройте SteamVR
5. Запустите IL-2. В случае успеха вы увидите преобразование изображения VorpX и обучающий материал
   
6. Наслаждайтесь!

*Убедитесь, что VorpX работает со SteamVR. Если нет, попробуйте другие "Device Selection" в "General"*

### Без SteamVR
*Эта глава не тестировалась автором. Вы можете помочь, отправив отчеты.*

Этот путь предназначен для повышения производительности за счет прямого перевода IL-2 на VR-гарнитуру.

#### Что нам нужно?
1. Программа для перевода вывода с ПК на VR-монитор. *Это может быть любая известная вам программа, например, Virtual Desktop*
2. VorpX. *Вы можете скачать для пробы версию с [4PDA](https://4pda.to/stat/go?u=https%3A%2F%2Fdisk.yandex.ru%2Fd%2F6iYhJOBe3ZtYAg&e=110018648&f=https%3A%2F%2F4pda.to%2Fforum%2Findex.php%3Fshowtopic%3D1034301%26st%3D5260%23entry118062338)*

Дополнительно: профиль IL-2, созданный сообществом на [форуме IL-2](https://forum.il2sturmovik.com/topic/69192-for-rift-s-il2_1946_bat-v385_vorpx_opentrack_logitech-x52_settings_v01/)

#### Настройка
1. Откройте VorpX. Вам нужно выбрать страницу конфигурации ("vorpX Config"). В разделе "General" выберите нужный вариант в "Device Selection", в зависимости от вашей VR-гарнитуры. Исключите SteamVR. Если ничего не работает, попробуйте последние два варианта
2. В разделе "Cloud/Local Profiles" скачайте онлайн или установите созданный сообществом профиль для IL-2
3. Откройте программу для ретрансляции (обычно как на ПК, так и на VR-гарнитуре)
4. Запустите IL-2. В случае успеха вы увидите преобразование изображения VorpX и обучающий материал

5. Наслаждайтесь!

*Убедитесь, что VorpX работает в фоновом режиме. Если нет, попробуйте другие "Device Selection" в "General"*
